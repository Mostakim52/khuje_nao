Map<String, Map<String, String>> localizedValues
final

A map that stores localized strings for different languages.

The keys are language codes (e.g., 'en' for English, 'bd' for Bengali), and the values are maps of key-value pairs where the key is a string representing a specific item or message, and the value is the localized string for that item in the corresponding language.
